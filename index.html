Logo Assets

PLEASE READ

The logo assets are available from Github as a download. Your download will contain both print and web versions of the assets. Use the appropriate folder so that you receive the correct assets. If you use the web assets in a print document, the colors will not appear correctly. The web versions should only be used when required for the church website / social media.

Font Usage

Homenaje is the new offical font for Church on the Hill. This font is an open-source font available from Google Fonts. It can be used on the web or desktop publication purposes. It should be used for all heading items.

Font Pairing

Please pair Homenaje with a sans-serif font. We strongly recommend pairing with Open Sans. Open Sans is also an open-source font for print and web use.
